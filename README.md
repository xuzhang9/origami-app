# Origami Learning App

A kid-friendly web app for learning origami with AI-powered search and step-by-step instructions.

## Features

- **AI-Powered Search**: Find origami tutorials using natural language queries
- **Step-by-Step Instructions**: Clear, easy-to-follow instructions with images
- **Device Authentication**: Secure family-only access
- **Progressive Web App**: Install on iPad/tablet for app-like experience
- **Offline Support**: Access cached origami tutorials without internet
- **Kid-Friendly UI**: Colorful, simple interface designed for children

## Tech Stack

- **Frontend**: Next.js 14+ (React)
- **Styling**: Tailwind CSS
- **Database**: Vercel Postgres
- **AI**: OpenAI GPT-4o
- **Hosting**: Vercel
- **PWA**: next-pwa

## Getting Started

See [IMPLEMENTATION_PLAN.md](./IMPLEMENTATION_PLAN.md) for detailed implementation steps.

### Prerequisites

- Node.js 18+ installed
- GitHub account
- Vercel account (free)
- OpenAI API key

### Quick Start

```bash
# Install dependencies (once project is set up)
npm install

# Set up environment variables
cp .env.example .env.local
# Edit .env.local with your API keys

# Run development server
npm run dev

# Open http://localhost:3000
```
